## 🔍 Recherche et Filtrage

### Barre de Recherche Générale

**Accessible depuis :** Presque toutes les pages

**Cherche dans :**
- 📖 Titres de publications
- 👤 Noms de chercheurs
- 🏷️ Mots-clés
- 📝 Descriptions

**Comment utiliser :**
```
Entrez un terme (ex: "agent-based", "simulation", "climate")
Appuyez sur Entrée ou cliquez sur la loupe
Les résultats s'affichent instantanément
```

### Filtres Avancés

#### Sur les Publications
- **Axe thématique** : Sélectionnez un ou plusieurs axes
- **Année** : Spécifiez une période (2020-2025)
- **Accès** : Public, protégé, privé
- **Chercheur** : Par auteur spécifique

#### Sur l'Équipe
- **Centre** : Filtrez par localisation
- **Rôle** : Chercheur, doctorant, directeur, etc.
- **Axe** : Publications et thèmes associés

#### Sur les Projets
- **Centre** : Projets par localisation
- **Axe thématique** : Filtrage par domaine
- **Statut** : En cours, achevés, à venir

### Tri des Résultats

**Options disponibles :**
- 📅 **Par date** : Plus récent au plus ancien
- 🔤 **Par titre** : Alphabétique
- 📊 **Par pertinence** : Selon votre recherche
- 👥 **Par auteur** : Nombre de publications
- ⬇️ **Par popularité** : Les plus téléchargés

## 👤 Espace Personnel

### Avant de se Connecter

- Vous avez accès à tous les contenus **publics**
- Les ressources protégées affichent une icône 🔒
- Cliquez sur "Connexion" pour accéder à plus

### Création de Compte

**Étapes :**
1. Cliquez sur **"Connexion"** en haut à droite
2. Sélectionnez **"Créer un compte"**
3. Entrez votre :
   - 📧 Email professionnel
   - 🔐 Mot de passe sécurisé (min. 8 caractères)

4. Validez votre email (lien d'activation)

- ❌ **Déconnexion** : Fermer votre session

### Navigation Efficace

**Pour trouver rapidement :**

- 🏷️ Exploitez les **filtres** pour affiner vos résultats
- ⭐ Sauvegardez les liens des pages utiles
- 🔖 Utilisez les signets de votre navigateur

### Optimisation Mobile

**Sur téléphone/tablette :**
- 👆 Tapez deux fois pour zoomer sur une image
- 📐 Tournez votre appareil en mode paysage pour plus d'espace
- 🔄 Les menus s'ouvrent/ferment automatiquement
- 📱 Utilisez le navigateur en mode "bureau" si besoin

### Téléchargement de Ressources

**Pour les datasets :**
1. Ouvrez la fiche du dataset
2. Cliquez sur **"Télécharger"**
3. Consultez la licence fournie
4. Créditez les auteurs dans vos utilisations

**Pour les publications :**
1. Cliquez sur le **lien DOI** (si disponible)
2. Ou téléchargez l'**archive complète**
3. Utilez les citations fournies (APA ou BibTeX)

### Citations Scientifiques

**Format disponibles :**

**APA :**
```
Auteur, A. (2023). Titre de la publication. Journal, 123(4), 567-890.
```

**BibTeX :**
```
@article{auteur2023,
  title={Titre},
  author={Auteur},
  journal={Journal},
  year={2023}
}
```

**Comment copier :**
1. Ouvrez la publication
2. Cliquez sur le format désiré
3. Copiez le texte
4. Collez-le dans votre gestionnaire de références

## ❓ FAQ

### Comptes et Connexion

**Q: J'ai oublié mon mot de passe, que faire ?**
R: Cliquez sur "Connexion" > "Mot de passe oublié" > Entrez votre email > Vous recevrez un lien de réinitialisation.

**Q: Puis-je utiliser le site sans créer de compte ?**
R: Oui ! Vous avez accès à tout le contenu public. Un compte permet d'accéder aux ressources protégées.

**Q: Comment supprimer mon compte ?**
R: Contactez l'équipe support via le formulaire de contact ou écrivez à [email support].

---

### Recherche et Filtrage

**Q: Comment chercher des publications d'un auteur spécifique ?**
R: Utilisez la barre de recherche avec le nom de l'auteur, ou filtrez par chercheur.

**Q: Puis-je combiner plusieurs filtres ?**
R: Oui ! Vous pouvez sélectionner plusieurs axes, années, et auteurs simultanément.

**Q: Pourquoi certaines publications ne s'affichent pas ?**
R: Elles peuvent être privées ou protégées. Connectez-vous ou contactez l'auteur pour y accéder.

---

### Téléchargements

**Q: Comment télécharger un dataset ?**
R: Ouvrez le dataset > Cliquez sur "Télécharger" > Le fichier se télécharge automatiquement.

**Q: Quelle est la licence des ressources ?**
R: Chaque ressource indique sa licence. Lisez-la avant utilisation.

**Q: Je dois créditer l'auteur, comment faire ?**
R: Utilisez le format de citation fourni sur la fiche de la ressource.

---

### Fonctionnalités

**Q: Comment accéder au globe interactif des centres ?**
R: Visitez l'accueil (/) > Section "Centres" > Explorez la carte interactive.

**Q: Puis-je cliquer sur les centres du globe ?**
R: Oui ! Passez la souris, cliquez pour voir les détails du centre.

**Q: Quelles sont les différences entre "public", "protégé" et "privé" ?**
R: 
- 🔓 **Public** : Accessible à tous
- 🔐 **Protégé** : Nécessite une authentification
- 🔒 **Privé** : Réservé aux administrateurs

---

### Support et Aide

**Q: Comment contacter l'équipe UMMISCO ?**
R: Visitez la page Contacts ou consultez l'annuaire pour les contacts directs.

**Q: Y a-t-il une API disponible ?**
R: Consultez la documentation ou contactez l'équipe technique pour plus d'informations.

### Technique

**Q: Quel navigateur dois-je utiliser ?**
R: Chrome, Firefox, Safari ou Edge (versions récentes). Le site fonctionne aussi sur mobile.

**Q: Le site fonctionne-t-il hors ligne ?**
